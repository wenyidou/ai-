这是一个浏览器插件，目前国内网站存在大量广告以及对网站阅读设置不利于浏览网页，降低网页的可读性。
随着ai时代的到来，我们无法实时对网页与ai交互，插件将集成ai和笔记功能方便浏览网页中的记录和分析。

使用教程：
1、打开Chrome浏览器或edge浏览器
2、打开插件内容部分，打开开发者模式
3、点击加载未打包的扩展程序
4、将解压好的安装包导入即可

ai功能的使用：
编辑content.js里面放着关于ai配置的功能代码，更新自己的即可
